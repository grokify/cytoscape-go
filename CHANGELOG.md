## [v0.1.0] - 2026-04-12

### Highlights

- Initial release of Go library for generating Cytoscape.js graph visualizations

### Added

- Graph builder with nodes, edges, and metadata ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
- Multiple layout algorithms: Cose, Dagre, Cola, Circle, Grid, Concentric, BreadthFirst, Preset ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
- Pre-built stylesheets: DefaultStyle, CodeGraphStyle, ERDStyle ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
- Standalone HTML generation with embedded JavaScript ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
- Interactive features: search, filters, export (PNG/SVG), layout selector ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
- Dark mode support ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
- JSON export in Cytoscape elements format ([`e461489`](https://github.com/grokify/cytoscape-go/commit/e461489))
